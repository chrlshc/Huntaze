# Cloudflare DNS configuration for huntaze.com with Netlify

## Record 1 - Apex Domain
Type: CNAME
Name: @ (or huntaze.com)
Target: apex-loadbalancer.netlify.com
Proxy: ON (orange)
TTL: Auto

## Record 2 - WWW Subdomain
Type: CNAME
Name: www
Target: @
Proxy: ON (orange)
TTL: Auto

## SSL/TLS configuration in Cloudflare
1. Go to SSL/TLS → Overview
2. Select: Full (strict)

## After deploying to Netlify
1. In Netlify → Domain settings
2. Add custom domain → huntaze.com
3. Netlify will automatically verify DNS
