## Key Features Documented

### Webhook Endpoint
- POST /api/webhooks/instagram - Receive webhook events
- GET /api/webhooks/instagram - Verification challenge
- Signature verification using HMAC SHA-256
- Asynchronous event processing
- Immediate 200 response (Meta requirement)

### Event Types
- media - New Instagram posts
- comments - New comments on posts
- mentions - Mentions in stories/posts

### Security
- HMAC SHA-256 signature verification
- Timing-safe comparison
- Environment-based secrets
- Signature header: x-hub-signature-256

### Environment Variables
- INSTAGRAM_WEBHOOK_SECRET - App secret from Meta
- INSTAGRAM_WEBHOOK_VERIFY_TOKEN - Verification token
