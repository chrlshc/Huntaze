Implements Tasks 9 & 10 from social-integrations spec

## Task 9: Instagram OAuth Flow

### Instagram OAuth Service
- Facebook OAuth 2.0 integration for Instagram Business/Creator accounts
- Long-lived tokens (60 days) with automatic refresh
- Page mapping (Facebook Page → Instagram Business Account)
- Business/Creator account validation
- Complete token lifecycle management

### OAuth Endpoints
- GET /api/auth/instagram - OAuth initialization with CSRF protection
- GET /api/auth/instagram/callback - OAuth callback with validation
- State parameter validation for security
- Automatic short-lived → long-lived token conversion
- Rich metadata storage (Page ID, IG Business ID, stats)

### Connect Page
- Beautiful gradient UI (purple → pink → orange)
- Requirements and permissions display
- Success/error message handling
- Help links for account conversion

## Task 10: Instagram Publishing

### Instagram Publish Service
- Create media containers (IMAGE, VIDEO, CAROUSEL)
- Poll container status until finished
- Publish containers to Instagram
- Get published media details
- Complete publish flows with error handling

### Publish Endpoint
- POST /api/instagram/publish - Publish content to Instagram
- Supports photos, videos, and carousels
- Auto-refreshes tokens
- Comprehensive error handling
- Returns published media details

## Features

### OAuth Flow
✅ CSRF protection via state parameter
✅ AES-256-GCM token encryption
✅ HTTP-only secure cookies
✅ HTTPS-only redirects
✅ Business/Creator account validation

### Publishing
✅ Single media (photos, videos)
✅ Carousels (2-10 items, mixed media)
✅ Container status polling
✅ Automatic token refresh
✅ User-friendly error messages

## API Examples

### Publish Photo
```json
POST /api/instagram/publish
{
  "mediaType": "IMAGE",
  "mediaUrl": "https://example.com/photo.jpg",
  "caption": "My photo #instagram"
}
```

### Publish Carousel
```json
POST /api/instagram/publish
{
  "mediaType": "CAROUSEL",
  "children": [
    { "mediaType": "IMAGE", "mediaUrl": "https://..." },
    { "mediaType": "VIDEO", "mediaUrl": "https://..." }
  ],
  "caption": "My carousel"
}
```

## Error Handling

- 400: Invalid media URL or format
- 401: Token refresh failed
- 403: Permission denied
- 404: Account not connected
- 408: Processing timeout
- 429: Rate limit exceeded
- 500: Server error

- 6 Instagram permissions requested
- 60-day token lifetime
- Token refresh once per day
- No refresh token (uses token refresh)

### Publishing
- Container-based publishing
- Status polling (30 attempts, 2s interval)
- Supports all Instagram media types
- Automatic error recovery
- Rate limit handling
