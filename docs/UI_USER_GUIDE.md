## Theme System

### Overview

Huntaze now supports three theme modes to match your preference and environment:
- **Light Mode**: Bright, clean interface for well-lit environments
- **Dark Mode**: Easy on the eyes for low-light conditions
- **System Mode**: Automatically matches your device's theme preference

### Switching Themes

#### Method 1: Theme Toggle (Recommended)

1. Look for the theme toggle in the top-right corner of the navigation bar
2. Click on your preferred theme:
   - ‚òÄÔ∏è **Light**: Bright theme
   - üåô **Dark**: Dark theme
   - üíª **System**: Follows your device

#### Method 2: Keyboard Shortcut

- Press `Ctrl/Cmd + Shift + T` to cycle through themes (coming soon)

### Theme Features

#### Automatic System Detection

When you select "System" mode:
- The theme automatically matches your operating system preference
- Changes when you switch your OS theme
- No manual switching needed

#### Persistent Preference

Your theme choice is saved automatically:
- Remembered across sessions
- Syncs across tabs
- No need to re-select each time

#### Smooth Transitions

Theme changes are smooth and pleasant:
- 200ms transition time
- No jarring flashes
- Maintains visual continuity

### Dark Mode Benefits

- **Reduced Eye Strain**: Easier on your eyes in low-light conditions
- **Better Battery Life**: Uses less power on OLED screens
- **Professional Look**: Modern, sleek appearance
- **Focus**: Less distraction from bright backgrounds

### Troubleshooting

**Theme not changing?**
- Clear your browser cache
- Check if JavaScript is enabled
- Try refreshing the page

**System theme not working?**
- Ensure your OS supports dark mode
- Check browser permissions
- Update your browser to the latest version

## Mobile Experience

### Overview

Huntaze is fully optimized for mobile devices with touch-friendly interfaces and mobile-specific features.

### Mobile Navigation

#### Bottom Navigation Bar

On mobile devices (< 992px width), you'll see a fixed bottom navigation bar with:
- Home
- Dashboard
- Messages
- Profile

**Features**:
- Always accessible (fixed position)
- Touch-friendly targets (44√ó44px minimum)
- Active state indication
- Smooth transitions

### Mobile-Optimized Features

#### Responsive Tables

Tables automatically convert to cards on mobile:
- Easier to read on small screens
- Swipe to scroll horizontally
- All data remains accessible

#### Full-Screen Modals

Modals take up the full screen on mobile:
- Better focus
- Easier interaction
- No accidental dismissal

#### Swipe Gestures

Swipe left on list items to reveal actions:
- Delete
- Archive
- More options

**Tip**: Swipe slowly to see the action button appear!

#### Optimized Forms

Forms are mobile-friendly:
- Larger input fields (48px height)
- Appropriate keyboards (numeric for numbers, email for emails)
- Auto-complete enabled
- Proper spacing between fields

### Mobile Tips

- **Landscape Mode**: Rotate your device for a wider view
- **Zoom**: Pinch to zoom on charts and images
- **Pull to Refresh**: Pull down on most pages to refresh data
- **Swipe Navigation**: Swipe right to go back (browser dependent)

## Animations

### Overview

Huntaze features smooth, professional animations that enhance the user experience without being distracting.

### Animation Types

#### Page Transitions

When navigating between pages:
- Smooth fade and slide effect
- 300ms duration
- Maintains context

#### Micro-Interactions

Small animations on interactive elements:
- **Buttons**: Scale up on hover (105%), down on click (95%)
- **Cards**: Subtle lift on hover
- **Links**: Smooth color transitions

#### List Animations

Lists appear with stagger effect:
- Items appear one by one
- 100ms delay between items
- Creates a flowing entrance

#### Scroll Reveal

Content reveals as you scroll:
- Triggers when 30% visible
- Animates only once
- Smooth fade and slide

#### Loading States

Skeleton screens while content loads:
- Shimmer effect
- Matches content structure
- Better than spinners

### Reduced Motion

If you prefer minimal animations:

1. **Enable in your OS**:
   - **macOS**: System Preferences ‚Üí Accessibility ‚Üí Display ‚Üí Reduce motion
   - **Windows**: Settings ‚Üí Ease of Access ‚Üí Display ‚Üí Show animations
   - **iOS**: Settings ‚Üí Accessibility ‚Üí Motion ‚Üí Reduce Motion
   - **Android**: Settings ‚Üí Accessibility ‚Üí Remove animations

2. **Result**: Huntaze will automatically disable animations

### Animation Tips

- **Performance**: Animations run at 60fps for smoothness
- **Battery**: Minimal impact on battery life
- **Accessibility**: All animations respect reduced motion preferences

## Accessibility

### Overview

Huntaze is built with accessibility in mind, ensuring everyone can use the platform effectively.

### Features

#### Keyboard Navigation

Navigate without a mouse:
- **Tab**: Move forward through interactive elements
- **Shift + Tab**: Move backward
- **Enter/Space**: Activate buttons and links
- **Escape**: Close modals and dropdowns
- **Arrow Keys**: Navigate within components

#### Screen Reader Support

Compatible with popular screen readers:
- NVDA (Windows)
- JAWS (Windows)
- VoiceOver (macOS/iOS)
- TalkBack (Android)

**Features**:
- Proper ARIA labels
- Semantic HTML
- Live regions for dynamic content
- Skip navigation links

#### Color Contrast

All text meets WCAG AA standards:
- Minimum 4.5:1 contrast for normal text
- Minimum 3:1 contrast for large text
- Works in both light and dark modes

#### Touch Targets

All interactive elements are touch-friendly:
- Minimum 44√ó44px size
- Adequate spacing between elements
- Clear focus states

### Accessibility Tips

- **Zoom**: Use browser zoom (Ctrl/Cmd + Plus/Minus)
- **Font Size**: Adjust in browser settings
- **High Contrast**: Use OS high contrast mode
- **Voice Control**: Compatible with voice control software

## Frequently Asked Questions

### General

**Q: How do I change the theme?**  
A: Click the theme toggle in the top-right corner and select your preferred theme.

**Q: Will my theme choice be saved?**  
A: Yes, your preference is automatically saved and persists across sessions.

**Q: Can I use keyboard shortcuts?**  
A: Yes, all interactive elements are keyboard accessible. Press Tab to navigate.

### Mobile

**Q: Why does the navigation look different on mobile?**  
A: On mobile, we use a bottom navigation bar for easier thumb access.

**Q: How do I delete items on mobile?**  
A: Swipe left on the item to reveal the delete button.

**Q: Can I use the app offline?**  
A: Some features work offline, with full offline support coming soon.

### Accessibility

**Q: Is the app screen reader compatible?**  
A: Yes, we support all major screen readers with proper ARIA labels.

**Q: Can I disable animations?**  
A: Yes, enable "Reduce Motion" in your OS accessibility settings.

**Q: What about color blindness?**  
A: We use patterns and labels in addition to colors for important information.
