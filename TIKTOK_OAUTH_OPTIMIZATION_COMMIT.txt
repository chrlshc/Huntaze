## Features

### 1. Error Handling
- 8 typed error codes (NETWORK_ERROR, TIMEOUT_ERROR, INVALID_TOKEN, etc.)
- Correlation IDs for request tracing
- Retryable vs non-retryable error distinction
- TikTok log IDs included in errors

### 2. Retry Strategy
- Automatic retry with exponential backoff
- Max 3 attempts with delays: 100ms ‚Üí 200ms ‚Üí 400ms
- Retry on: network errors, timeouts, rate limits, 5xx errors
- Skip retry on: invalid credentials, validation errors
