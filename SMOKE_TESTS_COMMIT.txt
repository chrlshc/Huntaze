## Test Separation
```
Vitest (npm test):
  ✅ Unit tests
  ✅ Integration tests
  ✅ Type validation tests
  ✅ Fast (< 1 minute)
  ✅ No server required

Playwright (npm run e2e:smoke):
  ✅ End-to-end tests
  ✅ Smoke tests
  ⚠️  Requires running server
  ⚠️  Slower (1-2 minutes)
```

## Usage
```bash
# Automated (recommended)
./scripts/run-smoke-tests.sh

# Manual
npm run dev &
npx wait-on http://localhost:3000
npm run e2e:smoke
```
