Implements Task 9 from social-integrations spec: Instagram OAuth Flow

## What's New

### Instagram OAuth Service
- Created InstagramOAuthService with Facebook OAuth 2.0 integration
- Supports Instagram Business/Creator accounts only
- Long-lived tokens (60 days) with automatic refresh
- Page mapping (Facebook Page → Instagram Business Account)
- Full account details retrieval (followers, media count, etc.)

### OAuth Endpoints
- GET /api/auth/instagram - OAuth initialization with CSRF protection
- GET /api/auth/instagram/callback - OAuth callback with validation
- State parameter validation for security
- Automatic token conversion (short-lived → long-lived)
- Instagram Business/Creator account validation

### UI Components
- Instagram connect page with gradient design
- Requirements and permissions display
- Success/error message handling
- Loading states and user feedback
- Help links for account conversion

## Security Features
✅ CSRF protection via state parameter
✅ AES-256-GCM token encryption at rest
✅ HTTP-only secure cookies
✅ HTTPS-only OAuth redirects
✅ Comprehensive error handling

## Technical Details
- Uses Facebook Graph API v18.0
- Requests 6 Instagram permissions
- Stores metadata (Page ID, IG Business ID, username, stats)
- Integrates with existing tokenManager and tokenEncryption services
- Follows TikTok OAuth patterns for consistency
