# Cloudflare DNS configuration for huntaze.com with Vercel

## 1. DNS records to add in Cloudflare:

### Record 1 - Root domain
Type: CNAME
Name: @ (or huntaze.com)
Target: cname.vercel-dns.com
Proxy: ON (orange)
TTL: Auto

### Record 2 - Subdomain www
Type: CNAME
Name: www
Target: @
Proxy: ON (orange)
TTL: Auto

## 2. In Cloudflare SSL/TLS:
- Mode: Full (strict)

## 3. After deploying on Vercel:
1. Go to vercel.com/dashboard
2. Select your project
3. Settings → Domains
4. Add → huntaze.com
5. Add → www.huntaze.com
6. Vercel may provide a TXT record if needed

## 4. Verification:
dig +short huntaze.com CNAME
curl -I https://huntaze.com
