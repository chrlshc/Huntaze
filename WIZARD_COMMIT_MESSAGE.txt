feat: Add 4-step setup wizard with Shopify-style dark UI

Implements a complete onboarding wizard that configures AI services
in under 30 seconds based on user selections.

## Features

### UI Components
- Dark Shopify-style backdrop with violet/pink radial glows
- 4-step wizard: Welcome → Platform → Goal → Tone
- Progress indicator with smooth transitions
- Mobile responsive and fully accessible (WCAG 2.1 AA)
- Layered card depth effect
- Gradient buttons and active states

### Backend Automation
- Service activation based on platform (OnlyFans, Instagram, TikTok, Reddit)
- Template provisioning for quick starts
- Dashboard configuration based on goals (Grow, Automate, Content, All)
- AI prompt generation based on tone (Playful, Professional, Casual, Seductive)
- Analytics tracking (completion time, skip rate, platform distribution)

### API
- POST /api/onboarding/wizard endpoint
- Service activation logic
- Template mapping
- Dashboard configuration
- AI config generation
- Database persistence

### Database
- user_wizard_completions table
- Analytics view for reporting
- Helper functions for queries
- Indexes for performance
